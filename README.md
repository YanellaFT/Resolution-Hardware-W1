# Resolution-Hardware W1
This is an RC Circuit built during week 1 of Resolution-Hardware!

The circuit includes 5 LED's that light up in a pattern. Each resistor attached to each LED has a different value, which allows the current to reach the LED at different speeds. This causes the lights to turn on at different times. 

The resistor in the beginning of the circuit, by the switch, controls how fast the lights turn on once the switch is closed. 

The LEDs also turn off in the opposite pattern they turn on in. Each capacitator attached to each LED has a different value, which allows the voltage to be stored in different amounts. This causes the lights to dim at different speeds.
